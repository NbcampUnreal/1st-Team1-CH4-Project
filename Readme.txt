# (프로젝트명)

## 🎮 개요

본 프로젝트는 **캐주얼 파티 액션 게임**으로, 2:2 팀플레이 기반의 장애물 경주 형식을 채택했습니다. 
플레이어는 제한된 시간 동안 상대 팀의 트랙에 다양한 방해 요소를 설치하고, 이후 자신의 트랙을 빠르게 통과해 결승점에 도달하는 것을 목표로 합니다.

## 🧩 주요 특징

- 2:2 팀플레이: 협동과 경쟁이 공존하는 파티 게임 구조
- 방해 기믹 설치 시스템: 게임 시작 전, 상대 팀 트랙에 방해 요소를 설치해 전략적 재미 강화
- 다양한 기믹 오브젝트: 점프대, 미끄럼 발판, 부스터 발판, 폭탄 트랩 등
- 카메라 구간 뷰 시스템: 방해 구간을 카메라로 순차적으로 비추며 전략적으로 방해 요소 배치 가능
- 비주얼 요소: 캐주얼하고 유쾌한 디자인으로 누구나 쉽게 접근 가능

## 🕹️ 게임 진행 방식

1. 로비: 플레이어들이 접속하여 게임을 시작
2. 준비 단계 (20초):
   - 각 팀은 상대 트랙의 지정 구간에 방해 기믹 설치
   - 카메라 버튼으로 구간을 전환하며 기믹을 전략적으로 배치
3. 경주 시작:
   - "3, 2, 1 Start!"와 함께 동시에 양 팀 경주 시작
   - 설치된 기믹을 피해 빠르게 결승점을 향해 질주
4. 승리 조건:
   - 먼저 완주한 팀이 승리!
